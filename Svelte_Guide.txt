SVELTE
1 Create a new Svelte project
	
	npx degit sveltejs/template frontend && cd frontend && npm install && npm run dev
	
	npm create svelte@latest my-app  -- to use latest svelte
	
	npx sv create my-app


2 Navigate into the project folder
	
	cd project_name

3 Install dependencies

	npm install

4 Start the development server

	npm run dev
